THE CONCEPT

Expert chess apears as a standard chess game - Except for some fields having red dots on it. Red dots mark fields which contain minigames. If a player hits these fields, a minigame will begin. You'd be well advised to win it and gain awesome items. Otherwise, your token will be dead!
